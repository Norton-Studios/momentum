Connect Your Development Tools

Link Momentum to your version control and development platforms to start collecting productivity metrics. At least one VCS provider is required.

Version Control Systems Required

GitHub
Not Connected

Connect to GitHub to track repositories, commits, pull requests, reviews, and contributors. Provides comprehensive metrics for code collaboration and delivery velocity.

Generate a token at github.com/settings/tokens
Required scopes: repo, read:org, read:user
Enter your GitHub organization name. For multiple organizations, separate with commas.
GL
GitLab
Not Connected

Connect to GitLab (cloud or self-hosted) to collect data from projects, merge requests, CI/CD pipelines, and contributors.

Use https://gitlab.com for GitLab.com or your self-hosted instance URL
Generate a token in GitLab Settings → Access Tokens with api scope
Enter group ID to import all projects, or comma-separated project IDs

CI/CD Platforms

J
Jenkins
Not Connected

Track pipeline executions, build success rates, duration metrics, and failure analysis from your Jenkins instance.

Generate an API token in Jenkins User Settings

Code Quality Tools

SQ
SonarQube
Not Connected

Import code quality metrics, test coverage, technical debt, security vulnerabilities, and code smells from SonarQube.

Generate a token in SonarQube → My Account → Security
Comma-separated list of SonarQube project keys to track
