Connect Your Tools

Momentum integrates with your existing development workflow. Connect at least one version control system to begin tracking metrics.

Version Control Required

⎈

GitHub

Connect your GitHub organization to import repositories, commits, pull requests, and contributor data. Track code collaboration and delivery velocity.

Not Connected
Generate at github.com/settings/tokens
Required scopes: repo, read:org, read:user
Enter your GitHub organization name. For multiple organizations, separate with commas.
⚡

GitLab

Connect to GitLab (cloud or self-hosted) to track projects, merge requests, CI/CD pipelines, and team activity across your development lifecycle.

Not Connected
Use https://gitlab.com for GitLab.com or your self-hosted instance URL
Generate a token in GitLab Settings → Access Tokens with api scope
Enter group ID to import all projects, or comma-separated project IDs
◆

Bitbucket

Connect Bitbucket to import repositories, commits, and pull requests from your workspace. Monitor code quality and team collaboration.

Not Connected
Generate an app password in Bitbucket Settings → App passwords
